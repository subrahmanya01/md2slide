md2slide

Version 1.0.0

md2slide is a lightning-fast Markdown presentation studio that runs entirely in your browser.

  • Write standard Markdown on the left.
  • Preview auto-scaling slides on the right.
  • Separate slides using --- on a new line.
  • Click Present to go fullscreen.

All data is temporarily stored in your local browser session.
